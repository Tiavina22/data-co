# Contrôle des Données Mobiles via API

## Description

Cette application permet de contrôler les données mobiles d'un appareil Android via une API REST. Elle utilise les commandes ADB pour activer ou désactiver les données mobiles depuis un navigateur web. Le projet est structuré avec un backend en Node.js utilisant Express et un frontend en HTML/CSS.

## Fonctionnalités

- **Activer les Données Mobiles** : Permet d'activer les données mobiles de l'appareil Android.
- **Désactiver les Données Mobiles** : Permet de désactiver les données mobiles de l'appareil Android.
- **Interface Utilisateur** : Interface simple avec boutons pour contrôler les données mobiles.
- **Gestion des Erreurs** : Retourne des messages d'erreur appropriés en cas de problème.

## Prérequis

- Node.js (v14 ou supérieur)
- ADB (Android Debug Bridge) installé et configuré
- Un appareil Android connecté via USB avec le débogage USB activé
